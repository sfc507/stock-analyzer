# 📈 台股觀測站：成交值與營收綜合分析

這是一個基於 Python 與 Streamlit 開發的台股分析工具。
本工具旨在協助投資人快速整理與篩選股票，透過將「成交值」與「營收成長率」進行交叉比對，並自動過濾特定產業，提供更清晰的選股視野。

## 🚀 功能特色

* **成交熱門綜合分析**：取成交值前 20 名個股，並自動關聯其單月營收年增率與產業類別。
* **營收飆股篩選**：針對全市場進行「單月營收年增率」排名（已自動剔除金融、營建等特定產業）。
* **數據自動清洗**：自動處理 CSV 匯出時常見的格式問題（如去除千分位逗號、修復 Excel 公式格式）。

## 📂 資料準備與匯出教學

本工具的資料來源為 [Goodinfo! 台灣股市資訊網](https://goodinfo.tw/tw/index.asp)。
請依照以下步驟匯出 **3 份必要的 CSV 檔案**：

### 前置作業
請先進入 [Goodinfo 智慧選股頁面](https://goodinfo.tw/tw/StockList.asp?RPT_TIME=&MARKET_CAT=%E6%99%BA%E6%85%A7%E9%81%B8%E8%82%A1&INDUSTRY_CAT=%E5%96%AE%E6%9C%88%E7%87%9F%E6%94%B6%E5%89%B5%E6%AD%B7%E5%8F%B2%E6%96%B0%E9%AB%98%40%40%E6%9C%88%E7%87%9F%E6%94%B6%E5%89%B5%E6%8E%92%E5%90%8D%E7%B4%80%E9%8C%84%40%40%E5%96%AE%E6%9C%88%E7%87%9F%E6%94%B6%E5%89%B5%E6%AD%B7%E5%8F%B2%E6%96%B0%E9%AB%98)。

### 1. 取得「成交值」資料
* 在頁面選單，設定資料顯示依據：
    * `交易狀況 - 成交資料` > `週` (或其他週期)
* 點選頁面下方的 **「匯出CSV」**。

### 2. 取得「單月營收年增率」資料
* 在頁面選單，設定資料顯示依據：
    * `月營收統計` > `月營收狀況` > `最新資料`
* 點選頁面下方的 **「匯出CSV」**。

### 3. 取得「產業別」資料
* 在頁面選單，設定資料顯示依據：
    * `公司基本資料` (通常位於基本分析類別下)
* 點選頁面下方的 **「匯出CSV」**。

---

## 💻 如何使用

1. 開啟本工具網頁。
2. 依序上傳上述三個 CSV 檔案。
3. 系統將自動執行模組化分析，並產生三個分頁報表供查閱。
